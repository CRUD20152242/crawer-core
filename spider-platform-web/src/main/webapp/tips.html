    文本抽取规则语法说明
  1. 目标标签有id 例如：
    < a href="#" id = "baidu">点我去百度< /a>
    < a href="#">点我去谷歌< /a> 只要在规则中输入 #baidu 即可通过id获取到文本“点我去百度”
  2. 目标标签有class 例如：
    < a href="#" class = "baidu">点我去百度< /a>
    < a href="#" class = "baidu">点我去谷歌< /a> 只要在规则中输入 .baidu 即可通过class获取到文本“点我去百度”,“点我去谷歌”
  3. 目标是标签例如：
    < a href="#" class = "baidu">点我去百度< /a>
    < a href="#" class = "baidu">点我去谷歌< /a> 只要在规则中输入 a 即可通过标签获取到文本“点我去百度”,“点我去谷歌”
  4. 目标是多级标签例如：
    < div>< div class="div1">< div id="div2">< a href="#" class = "baidu">点我去百度< /a>< /div>< /div>< /div>
    < a href="#" class = "baidu">点我去谷歌< /a> 只要在规则中输入 div .div1 #div2 a 即可通过组合规则获取到文本“点我去百度”。
  5. 目标是带有指定属性的标签：
    < a href="#" class = "baidu">点我去百度< /a>
    < a href="#" >点我去谷歌< /a> 只要在规则中输入 a.baidu 即可通过组合规则获取到文本“点我去百度”。

注意：此版本仅仅支持爬取指定网页的文本信息
